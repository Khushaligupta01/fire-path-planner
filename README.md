🔥 FIRE Path Planner
Your AI-Powered Financial Independence Advisor

Plan early. Retire free. Live fully.

FIRE Path Planner is an intelligent financial planning web app that helps users achieve Financial Independence, Retire Early (FIRE) through AI-driven insights, a personalized Money Health Score, and a year-by-year investment roadmap tailored to individual financial situations.

🎯 Problem Statement

Most people in India struggle with:

❓ Understanding how much money they need to retire early
📉 Evaluating whether their current financial health is strong
💰 Deciding where and how much to invest monthly

Traditional financial calculators only provide numbers — not actionable guidance.

💡 Solution

FIRE Path Planner combines:

Rule-based financial logic
AI-powered insights (LLM integration)

to deliver a complete, personalized financial strategy in minutes.

✨ Key Features
🔥 1. FIRE Path Planning
Calculates FIRE target using the 25× rule
Computes required monthly SIP
Estimates 6-month emergency fund
Adjusts for inflation (~5–6%)
Generates a year-by-year investment roadmap
💰 2. Goal-Based Planning
Add goals (house, car, education, travel)
Calculates SIP required for each goal
Combines FIRE + Goals into one unified plan
🛡️ 3. Insurance Recommendation
Life insurance = 10× annual income
Health insurance based on age group
Covers a commonly ignored financial gap
💯 4. Money Health Score (Core Differentiator)

Scores financial health out of 100 across 6 dimensions:

Dimension	What It Measures
🧰 Emergency Preparedness	Safety net availability
📈 Investment Discipline	Consistency of investing
💳 Debt Health	Debt vs income balance
🧾 Tax Efficiency	Use of tax-saving tools
🏖️ Retirement Readiness	FIRE progress
🛡️ Insurance Coverage	Risk protection
📊 5. Investment Roadmap Visualization
Interactive wealth growth charts
Year-by-year financial tracking
🎯 6. Smart Suggestions Engine
Identifies weakest financial areas
Provides targeted improvement tips
Mimics AI-driven personalization even without API calls
🤖 7. AI Insights (LLM Integration)
Explains financial plans in simple language
Highlights risks and opportunities
Provides personalized recommendations
🎉 8. Life Event Advisor (Bonus Feature)

Handles real-life scenarios like:

💸 Bonus received
💍 Marriage
🏠 Inheritance

Provides contextual financial guidance for each event.

🚀 Getting Started
✅ Prerequisites
Python 3.9+
OpenAI API Key (optional for AI insights)
⚙️ Installation
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/fire-path-planner.git

# 2. Navigate into the project
cd fire-path-planner

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the application
streamlit run main.py

👉 App will run at: http://localhost:8501
